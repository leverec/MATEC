## Installation & Usage
**1. Clone This Repository**
```bash
git clone https://github.com/leverec/MATEC.git
```
**2. Run The Script**
```bash
python main.py
```
**3. Type A Commands**

_Example:_
```text
$ circle r 7
Radius        : 7cm
Diameter      : 14cm
Circumference : 43.98cm
Area          : 153.94cm²
$ square s 6
Side      : 6cm
Diagonal  : 8.49cm
Perimeter : 24cm
Area      : 36cm²
$ rectangle b 8 15
Length    : 15cm
Width     : 8cm
Diagonal  : 17cm
Perimeter : 46cm
Area      : 120cm²
```
**4. Type exit to close the program**
```text
$ exit
```
For more syntax, you can read the documents that i made [Full Syntax Documentation](./Syntax.md)
